Discuss the scope of var, let, and const?

Var : The scope of a var variable is functional scope.

Let : The scope of a let variable is block scope.

Const : The scope of a const variable is block scope.

Tell us the use cases of null and undefined?

Undefined: Undefined means the variable has been declared, but its value has not been assigned.

Null : Null means an empty value or a blank value.

What do you mean by REST API?

Rest API : A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style.
